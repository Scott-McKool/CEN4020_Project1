Source code files:

main.py - handles the backend and game logic of the project, done in python.

Proj1.py - main GUI window source code, used to run the game and used when compiling to the .exe file

Operating System: Windows

Languages: Python

Python modules used: TKinter, simpleaudio (cexen fork), json, pyinstaller (check requirements.txt for external module installation)

Version control method: GitHub

IDLE: Visual Studio Code

Compilation commands:
Open terminal window in Proj1 folder

install pyinstaller using "pip install pyinstaller"

run "pyinstaller --onefile --windowed -w 'Proj1.py'

Proj1.exe gets created in /dist subfolder, bring Proj1.exe into the main Proj1 folder for it to work properly. (replace existing Proj1.exe if necessary)
